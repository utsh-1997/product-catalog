# Product Catalog App

A full stack product catalog with complete CRUD operations.

## Live Demo
- Frontend: https://product-catalog-eosin.vercel.app
- API: https://product-api-0t84.onrender.com/products.php

## Tech Stack
- Frontend: React, Vite
- Backend: PHP REST API
- Database: MySQL
- Deployment: Vercel (frontend), Render (backend)

## Features
- View all products
- Search/filter products in real time
- Add new products
- Edit existing products
- Delete products
